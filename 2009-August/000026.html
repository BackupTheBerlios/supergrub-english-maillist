Hi,

	As you might have noticed I am currently working at baells series of 
Super Grub Disk.
	The first thing that I have changed is that the baells which was the 
grub legacy based version that used Makefiles now it is going to be 
based on grub2.
	There is the baells directory for upstream supergrub 
(distro-independent) and there is the baellsdeb (or something similar) 
for saving the debian directory which let us make a Debian package.
(Wiki needs update on the baells meaning.)

	I have also uploaded these versions as tar.gz at files at berlios.

	The debian package is for Debian unstable.

	You are welcome to try the upstream version and the Debian package if 
you want to. And report back your experiences.

	I want to focus a bit more on Debian package development, i.e, knowning 
how to build sg2d for ppc, ieee..., and other architectures.

	After that we will be able to discuss how to use the LUA scripts 
(Grub2 scripting is based on LUA although some sh scripting is also 
available) in order to make a menu, with translation support and so on.

	Of course, if you want to work on the package side you are also invited 
to participate.

	Any comment is welcome.
