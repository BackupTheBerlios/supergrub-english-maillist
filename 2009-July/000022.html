Super Grub2 Disk 0.01 is available.

However what I want to tell you is that you can find information on how 
I have done it thanks to grub2 and Debian's grub2 at:
http://www.supergrubdisk.org/w/index.php5?title=SGDDevCamporells#How_to_modify_grub-rescue-pc_package

In the same wiki page (just above the section I have mentioned) you can 
see the grub.cfg contents that I use.

As you can see Super Grub2 Disk 0.01 has almost any features but just 
trying to boot Linux (with the ubuntu default kernel and initrd.img 
localisation).

The objective of Camporrells series is to make this grub.cfg a bit more 
complex at each release so that we can have as many features as former 
Super Grub Disk. Although, in theory, we won't be able to fix the boot 
of linux from SG2D.

If you ever had an unstable Debian installation and you wanted to 
reproduce the steps I have mentioned in the wiki it would be very great.
