Hi,

This is a message from Super Grub Disk Development Summer School ;).

This is your first task as a Super Grub Disk newbie:
    - Try to add an option called "Show release" under the Support menu.

Some clues:
              - tail_menu.lst on dev_sgd already does it show a file.
              - release is found at boot/sgd/release (or was it release.txt) ?
              - If it does not show anything try the catis on  (begin)
and catis off (end) commands.

In order to make it multilingual you will need to define a tr_
variable at dev_sgd/conf/translate.sh (maybe it is called
SXXtranslate.sh)

And the most important one: If you have any question just ask it.
