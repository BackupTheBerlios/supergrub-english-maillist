Lua was moved to grub-extras upstream so the code for lua is in the
latest debian unstable source package but isn't enabled. To build a
package with lua support:

sudo apt-get build-dep grub2
apt-get source grub2
cd grub2*
add "export enable_lua := yes" to debian/rules after enable_915resolution
fakeroot debian/rules binary

Whenever I get to making a ppa for SG2D I will include the latest
grub2 from debian unstable + lua support as well.
