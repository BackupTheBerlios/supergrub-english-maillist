Jordan Uggla escribió:
> Upstream Grub2 moved lua support to grub-extras, and while Debian does
> include some modules from grub-extras in their grub2 packages, they
> don't currently include lua. Here are the steps to create grub
> packages with lua support:
Hi Jordan,

These are the things you should include (or specify in the source code 
documentation if they are not included) when doing a release.
> 
> Get all the needed dependencies:
> 
> # apt-get install bzr fakeroot
> # apt-get build-dep grub2
> 
> Get the source package:
> 
> $ cd directory/you/want/to/build/package/in/
> $ apt-get source grub2
What's exactly the source grub2 version that you get.
> 
> Add lua to the source package:
> 
> $ cd grub2-*/
> $ bzr branch http://bzr.savannah.nongnu.org/r/grub-extras/lua/
> debian/grub-extras/lua/
What's exactly the bazaar revision that you are getting.

That's it!
