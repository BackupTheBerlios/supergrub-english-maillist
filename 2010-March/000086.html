There were many changes that needed to be made to the scripts to make
them appropriate (or even functional) for Super Grub2 Disk, for
instance the scripts originally looked for isos on the same drive as
grub was installed to, which would mean none would be found when
booting from super grub disk itself. I kept the files separate for
many of the changes to hopefully make them easier to follow and pick
out individually. So for commits 2f0a554 through 6e73464 the files
stayed separate. In commit bed67c3 I took the small amount of code
from listisos.lua (the real work was all in bootiso.lua), added it to
bootiso.lua, deleted listisos.lua, and renamed bootiso.lua to
isodetect.lua. So after bed67c3 there is only one file in the source
tree related to booting isos, and that file is isodetect.lua.

bootiso.lua and listisos.lua exists only in revision history.

>
> I mean... this is not Super Grub2 Disk source code, isn't it?

It wasn't, but now it is :). The only file left from this,
isodetect.lua, is now Super Grub2 Disk source code. If you want to
look at how it got there you need to look through the revision
history.
