Well, there is 2 times the following sentence in the docs:

"3. Select Umount. Do not select Safe Extraction!!!"

My suggestions for the docs:
If the sentence above is correct for some systems then:

"3. Select Umount. Do not select Safe Extraction!!! If you get error
messages with the grub commands below, leave the pendrive mounted (works for
Ubuntu 10.10)"

If the sentence above is not correct anymore just delete it and 
"1. Find your pendrive icon in your Desktop. 
 2. Right-click on it"
too.

After the grub commands insert

   1. Find your pendrive icon in your Desktop.
   2. Right-click on it
   3. Select Umount.

before "You can now unplug your pendrive. !"
