==First of all==

I don't mind the devices being discovered from /proc/partitions from 
/sys or from another place (even kernel calls).

name to rescue projects like: Rescatux, boot-repair, BIS, System Rescue

exist at all.

==What I mean== is that you should think on making the code to check for 
/sys if it is a better source and if /sys does not exist it should be 
able to read from /proc/partitions.
