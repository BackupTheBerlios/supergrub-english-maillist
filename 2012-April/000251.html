Author: Jordan Uggla <Jordan.Uggla at gmail.com>
Date:   Sun Apr 29 00:19:49 2012 -0700

    Change how the language selection menu is implemented
    
     * Move info_cfg files into menus/sgd_locale and change their format
     * slightly.
    
     * info_cfg files now contain a variable $lang_code, wich is set to the
     * two letter language code for the translation in question, and
     * $lang_name which is the name of the language in that language.
    
     * Language menu now uses the name of the language in that language,
     * rather than the name of the language in English.
    
     * Adding new translations is made easier by automating the creation of
     * new info_cfg files in grub-mkrescue whenever a new po file is
     * created. This way translators should hopefully not need to know
     * anything about Super GRUB2 Disks translation system other than how to
     * work with gettext.
    
     * Document how to add new translations in the new system in
     * TRANSLATION.
    
     * Remove set -x from grub-mkrescue to make it easier to notice message
     * to translators, and to make any error messages easier to see (so they
     * aren't lost in the flood of normal output).
