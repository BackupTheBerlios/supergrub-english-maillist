    Add Finnix ISO loopback support

    Finnix 107 includes findiso= support.  This change searches for Finnix,
    and will build the following menu items, depending on what is present:

      * 32-bit userland, 64-bit kernel (/boot/x86/linux64)
      * 32-bit userland, 32-bit kernel (/boot/x86/linux)
      * 64-bit userland, 64-bit kernel (/boot/amd64/linux)

    Note that the last option for Finnix 64-bit userland, which is not
    distributed directly by Finnix, but may be built manually by users.
