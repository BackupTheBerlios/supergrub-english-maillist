    Append serial terminal in "Enable serial terminal"
    
     * Before this commit we used "terminal_output console serial" when we
     * were "enabling" serial terminal, unfortunately if you happened to be
     * using an output/input other than console (like gfxterm) that command
     * would also disable that and switch to console instead (breaking
     * SG2D's rendering of non-ASCII characters). With this commit we
     * instead use "terminal_output --append serial; terminal_input --append
     * serial" which will preserve any existing terminals while adding
     * serial input/output.
